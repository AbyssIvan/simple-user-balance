# simple-user-balance

## Условия задачи
Реализация приложение для работы с балансом пользователей.

## Реализация
Реализовано REST API с возможностью добавления/редактирования/удаления пользователей.
Реализован функционал для:
- зачисления средств пользователю
- списание средств пользователя
- перевод средств между пользователями
- получать баланса пользователя
Хранение баланса реализована в центах/копейках, однако согласно условиям задачи в запросах суммы передаются в формате 000.00
Документация по API доступна по адресу http://localhost/api/documentation

## Валидация
Помимо валидации основных параметров, так же добавлена валидация негативного баланса на уровне доменной логики
